# granite-ml-coder — live demo

A fine-tuned **IBM Granite 3.1 1B** that writes Python ML code (scikit-learn,
pandas, NumPy, Keras) and explains pipeline concepts. Model:
[Tekimax/granite-ml-coder](https://huggingface.co/Tekimax/granite-ml-coder) ·
Workshop: [github.com/TEKIMAX/tekimax-ML-Model-workshop](https://github.com/TEKIMAX/tekimax-ML-Model-workshop)

> Runs on the **free CPU tier**, so responses take a little while. It's a 1B
> model — great for fast drafts, verify before use.
